# Gestion de Stock - React App

Application React pour gérer un inventaire de produits. Permet d'ajouter, modifier, supprimer et rechercher des produits.

## Fonctionnalités
- ✅ Ajouter, modifier, supprimer des produits
- ✅ Recherche par nom/catégorie
- ✅ Sauvegarde automatique (localStorage)
- ✅ Calcul du stock total
- ✅ Design responsive

## Installation
```bash
git clone https://github.com/ZahiraK/stock-manager.git
cd stock-manager
npm install
npm start
```
